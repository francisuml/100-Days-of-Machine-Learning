
# 📌 Day 4: Data Visualization with Matplotlib, Seaborn, and Plotly

## 📖 Overview
Welcome to **Day 4** of our **100-day Machine Learning journey**! Today, we focus on **Data Visualization**, a critical step in exploring and understanding datasets. We will use **Matplotlib, Seaborn, and Plotly** to create insightful visualizations.

✅ Compare insights from different visualization techniques

- **Seaborn**: Pair plots, heatmaps, box plots
- **Plotly**: Interactive scatter plots, bar charts, and line graphs
